# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Chef2Go Frontend

Overview
--------
React-based frontend for Chef2Go. Implements user flows for browsing recipes, placing orders, and accessing premium content.

Getting started
---------------
1. cd chef-2-go-frontend
2. npm install
3. npm start

Notes
-----
Ensure backend API endpoints are configured in environment variables for API calls.
